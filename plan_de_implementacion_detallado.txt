### Tarea 1.3: Reemplazar estilos CSS modules por clases de Bootstrap

- [ ] Importar y configurar fuente retro (como la que ya existe: retro_gaming.woff2)

## CRITERIOS DE ÉXITO

### Criterios Técnicos:
- [ ] Todas las páginas deben usar componentes de Bootstrap con estilo retro
- [ ] Las animaciones no deben afectar el rendimiento del juego
- [ ] El sistema debe mantener la funcionalidad existente
- [ ] El código debe seguir buenas prácticas de desarrollo

### Criterios de UX:
- [ ] La información debe ser clara y accesible
- [ ] Las decisiones deben tener feedback inmediato
- [ ] La estética retro debe ser consistente en toda la aplicación
- [ ] La experiencia debe ser más intuitiva y satisfactoria

### Criterios de Juego:
- [ ] Las mecánicas de juego deben mantenerse intactas
- [ ] La toma de decisiones debe ser más informada
- [ ] El progreso debe ser más visible y satisfactorio
- [ ] La experiencia debe ser más adictiva sin ser manipuladora
